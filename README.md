sbt-sbs
=======
Simple plug-in for uniform configuration of *Reactive Codes*  [sbt][1] projects.


### Supported [sbt][1] version:
-  0.13.5


## Artefact Coordinates:
`addSbtPlugin("codes.reactive.sbt" % "sbt-codes" % "0.1.0-SNAPSHOT")`
